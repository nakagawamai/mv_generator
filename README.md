# mAii MV Generator

単一 HTML で動作する、静止画ベースの MV 風ループ動画ジェネレーターです。  
ケン・バーンズ効果（ズーム/パン）と各種エフェクトを重ね、ブラウザ上で録画して書き出せます。

## 特徴

- 画像 1 枚からループ動画を生成
- ケン・バーンズ効果（ズーム、パン、ループ秒数）
- エフェクト切り替えと強度調整
  - 雨粒 / 光の粒 / 星くず / 霧 / 雲 / グレイン / 光の呼吸 / 揺らぎ
- 録画機能（`REC`）
- 環境によっては直接 MP4 録画
- webm で保存された場合はブラウザ内で MP4 変換

## ファイル構成

```text
.
├── mv_generator.html

```

## 使い方

1. `mv_generator.html` をブラウザで開く
2. 必要に応じて画像をドラッグ&ドロップ、または `画像変更` ボタンで差し替える
3. 右パネルでエフェクトとパラメータを調整する
4. `● REC` を押して録画開始（3 秒カウントダウン後に録画）
5. 録画終了後、ファイルが自動ダウンロードされる
6. 出力が webm の場合は `▶ MP4変換` で MP4 を生成

## 動作要件

- モダンブラウザ（Chrome / Edge / Safari の最新版推奨）
- `MediaRecorder` / `Canvas` / `VideoEncoder` が利用可能な環境
- MP4 変換時は `mp4-muxer` を CDN から読み込むためインターネット接続が必要

## 注意

- 出力形式はブラウザ実装に依存します。  
  - `video/mp4;codecs=avc1` が利用可能なら MP4 で直接保存  
  - 非対応時は webm 保存後、MP4 変換ボタンを利用
- 録画長は「ケン・バーンズ > 速さ（秒）」の値に連動します。
